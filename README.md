# Conhecendo o DOM 🧭✨
https://conhecendo-dom.vercel.app/

Um repositório com exemplos e exercícios para aprender a manipular o DOM (Document Object Model) usando JavaScript puro. Ideal para iniciantes que querem entender como selecionar elementos, escutar eventos e atualizar o conteúdo da página dinamicamente.

## 🚀 Sobre
Este projeto reúne pequenos exemplos e atividades práticas para:
- Selecionar elementos do DOM 🔎
- Alterar atributos, estilos e conteúdo ✏️
- Trabalhar com eventos (click, input, submit, etc.) 🖱️
- Criar, inserir e remover elementos dinamicamente ➕➖
- Praticar boas práticas e organizar código modularmente 🧩

## 📁 Estrutura sugerida do repositório
- index.html — página principal de exemplos
- /css — arquivos de estilo
- /js — scripts que manipulam o DOM
- /exercicios — exercícios com enunciados e soluções

Exemplo:
```html
<!-- index.html -->
<!doctype html>
<html>
  <head>
    <meta charset="utf-8" />
    <title>Conhecendo o DOM</title>
    <link rel="stylesheet" href="css/style.css" />
  </head>
  <body>
    <script src="js/main.js"></script>
  </body>
</html>
```

## ⚙️ Como usar (rápido)
1. Clone o repositório:
```bash
git clone https://github.com/andrMiotto/conhecendo-DOM.git
```
2. Abra `index.html` no navegador (duplo clique ou Live Server) 🖥️
3. Edite os arquivos em `js/` e atualize a página para ver as mudanças 🔁

## ✍️ Exemplos de exercícios
- Selecionar todos os itens de uma lista e aplicar uma classe quando clicados ✅
- Criar um formulário que adiciona cards à página sem recarregar 🔄
- Validar campos e mostrar mensagens inline 📝

## 💡 Dicas de estudo
- Use o DevTools do navegador para inspecionar o DOM e testar seletores 🧰
- Comece com `querySelector` / `querySelectorAll`, depois veja `getElementById` e `getElementsByClassName` 🔍
- Evite manipular o DOM em loops intensivos sem fragmentos (use DocumentFragment) ⚡
